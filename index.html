Get started

    API Endpoint Base

        https://gocampaign.herokuapp.com/api/v1

Campaign API ini digunakan untuk adminstrasi data campaign. Dari mulai membuat campaign, funding campaign, transaksi funding, dll.

Agar bisa menggunakan API ini diperlukan token dari JWT, yg sudah disediakan untuk masing-masing user. Token ini didapatkan setelah proses register user.

Informasi selengkapnya bisa menghubungi saya di kuswandi.ti@gmail.com

Desain & relasi tabel dapat dilihat dari image berikut :

Sebelum running, create database terlebih dahulu dengan nama db_campaign

Selanjutnya tabel akan dijalankan dengan automigrate

Register User


# Contoh Request Register User :
url
-- https://gocampaign.herokuapp.com/api/v1/users

Untuk proses register user, silahkan menggunakan method POST dengan url berikut :
https://gocampaign.herokuapp.com/api/v1/users 



Contoh Request :

{
  "name": "user",
  "occupation": "Pegawai Swasta nasional",
  "email": "user@mail.com",
  "password": "rahasia"
}
	
	
Response Sukses :

{
  "meta": {
    "message": "Account has been registered",
    "code": 200,
    "status": "success"
  },
  "data": {
    "id": 1,
    "name": "user",
    "occupation": "Pegawai Swasta nasional",
    "email": "user@mail.com",
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VyX2lkIjoxfQ.9PtESZsQIHTFSRBDXLWJsKjzPTZfZvO6LTnKNO3ClZw",
    "image_url": ""
  }
}


Response Gagal :

{
  "meta": {
    "message": "Register account failed.",
	"code": 422,
	"status": "error"
  },
  "data": {
    "errors": [
	  "Key: 'RegisterUserInput.Name' Error:Field validation for 'Name' failed on the 'required' tag"
	  ]
  }
}

QUERY PARAMETERS

Field Type Description Mandatory
name String Nama User Ya
occupation String Pekerjaan User Ya
email String Email User Ya
password String Password User Ya

Cek Available Email

url
-- https://gocampaign.herokuapp.com/api/v1/email_checkers

Untuk mencari apakah email bisa digunakan untuk register atau tidak, silahkan menggunakan method GET dengan url berikut :
https://gocampaign.herokuapp.com/api/v1/email_checkers

}
	
	
Response Sukses :

{
  "meta": {
    "message": "Email is available",
    "code": 200,
	"status": "success"
  },

	"is_available": true
  }
}

{
  "meta": {
    "message": "Email not available",
    "code": 422,
    "status": "error"
  },
  "data": {
    "errors": [
	  "Key: 'CheckEmailInput.Email' Error:Field validation for 'Email' failed on the 'required' tag"
    ]
  }
}

QUERY PARAMETERS

Field Type Description Mandatory

Membuat Campaign Baru


# Contoh Request Fetch Data Campaign :
url
-- localhost:6600/api/v1/campaigns

Untuk membuat data campaign baru, silahkan menggunakan method POST dengan url berikut :
https://gocampaign.herokuapp.com/api/v1/campaigns 



Contoh Request :

Header :
Authorization : Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VyX2lkIjoxfQ.9PtESZsQIHTFSRBDXLWJsKjzPTZfZvO6LTnKNO3ClZw

	"name": "Campaign Siswa Berprestasi",
	"short_description": "Penggalangan dana untuk siswa berprestasi",
	"description": "Kegiatan campaign yang diadakan untuk membantu para siswa yang berprestasi",
	"goal_amount": 10000000,
	"perks": "Nominal funding bebas, Bisa berkali-kali funding, Laporan funding transparan"
}	
	
Response Sukses :

{
  "meta": {
	"message": "Success to create campaign",
	"code": 200,
	"status": "success"
  },
  "data": {
	"id": 1,
	"user_id": 1,
	"name": "Campaign Siswa Berprestasi",
	"short_description": "Penggalangan dana untuk siswa berprestasi",
	"image_url": "",
	"goal_amount": 10000000,
	"current_amount": 0,
	"slug": "campaign-siswa-berprestasi-1"
  }
}


Response Gagal :

{
  "meta": {
	"message": "Unauthorized",
	"code": 401,
	"status": "error"
  },
  "data": null
}

QUERY PARAMETERS

Field Type Description Mandatory
name String Nama / Title Campaign Ya
short_description String Deskripsi singkat dari Campaign Ya
description String Deskripsi secara lengkap mengenai Campaign Ya
goal_amount String Jumlah rupiah yang akan jadi target dari Campaign Ya
perks String Point-point keunggulan dari Campaign (setiap point akan dipisahkan dengan tanda koma ,) Ya

Create Transaksi Funding Campaign


# Contoh Request Create Transaksi Funding Campaign :
url
-- https://gocampaign.herokuapp.com/api/v1/transactions

Untuk membuat transaksi baru funding campaign, silahkan menggunakan method POST dengan url berikut :
https://gocampaign.herokuapp.com/api/v1/transactions 



Contoh Request :

Header :
Authorization : Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1c2VyX2lkIjoxfQ.9PtESZsQIHTFSRBDXLWJsKjzPTZfZvO6LTnKNO3ClZw
{
  "amount": 5000000,
  "campaign_id": 1
}
	
Response Sukses :

{
  "meta": {
	"message": "Success to create transaction'",
	"code": 400,
	"status": "success"
  },
  "data": {
	"id": 1,
	"campaign_id": 1,
	"user_id": 1,
	"amount": 5000000,
	"status": "pending",
	"code": "",
	"payment_url": "https://app.sandbox.midtrans.com/snap/v3/redirection/7ae296ca-f2eb-42a4-a149-9fe72edb5330"
  }
}


Response Gagal :

{
  "meta": {
	"message": "Unauthorized",
	"code": 401,
	"status": "error"
  },
  "data": null
}

QUERY PARAMETERS

Field Type Description Mandatory
amount Int Jumlah Rupiah Funding Ya
campaign_id Int ID Dari Campaign yg Akan Kita Lakukan Funding Ya
